Class: Ball

Ball(positionX, positionY, radius, color)

new Ball(positionX, positionY, radius, color)

Clase pelota. Esta clase está preparada para ser dibujada en un canvas en un HTML, para ello tiene definida una posición X y una posicón Y necesarias para ubicarla en la web, tiene un radio para poder representar el tamaño de la esfera y un color que le dará un acabado atractivo a la pelota.
Parameters:
Name Type Default Description
positionX int 0 posición actual en el eje x del eje de coordenadas que contenga la pelota.
positionY int 0 posición actual en el eje y del eje de coordenadas que contenga la pelota.
radius int 1 radio de la pelota.
color string color de la pelota.

Class: ChessBoardQueens

ChessBoardQueens(chessBoardWidth, chessBoardLength, generalized)

new ChessBoardQueens(chessBoardWidth, chessBoardLength, generalized)

Constructor de un tablero que recibe 3 parámetros que lo definirán, siendo la altura, la anchura del mismo y la definición de si el problema es el general o el clásico.. El tablero contendrá como atributos una altura, una anchura, un conjunto de casillas, un contador de soluciones, un conjunto de soluciones y un booleano que define el tipo de problema. En el tablero de reinas en cada casilla habrá un booleano que nos indicará si hay una reina (true) o si no la hay (false).
Parameters:
Name Type Default Description
chessBoardWidth int 8 Anchura del tablero
chessBoardLength int 8 Altura del tablero
generalized bool true Define si el problema se va a resolver de manera clásica (false) o generalizada (true)

checkFreeDiag(chessBoard, currentRow, foundQueens) → {bool}

Función que comprueba si la reina de la columna foundQueens en la fila currentRow es amenazada diagonalmente por alguna de las reinas anteriormente colocadas. Para ello, para cada reina del tablero que ya ha sido colocada se avanza diagonalmente hacia la derecha tanto para arriba como para abajo buscando si se coincide en filas y columnas al coger la fila y columna de la dicha reina + las posiciones avanzadas diagonalmente hacia arriba y hacia abajo.

findQueens(chessBoard, foundQueens, queensToFind)

Función principal del programa. Se encarga de hacer la llamada recursiva cuando aún no se ha encontrado solución (no se han colocado 8 reinas) si se puede colocar una reina y es la encargada de ir eliminando las reinas ya comprobadas.
Parameters:
Name Type Description
chessBoard array El tablero está representado por una matriz de booleanos, donde un 1 indica que hay una reina y un 0 que no lo hay. Además, como en la implementación tenemos en consideración que en cada columna solo habrá una reina, sabremos siempre que desde la columna 0 hasta la columna foundQueens habrá exactamente el valor foundQueens de reinas encontradas.
foundQueens number Número de reinas ya encontradas al hacer la llamada
queensToFind number Número de reinas para el que hay que solucionar el problema

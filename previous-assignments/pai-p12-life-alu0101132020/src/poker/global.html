playGame(deck, results, numberHands, numberCardsPerHand, handPossibilities)

Función que hace el reparto de cartas de una partida y lleva el recuento de la mejor jugada de cada mano en el array de resultados.
Parameters:
Name Type Description
deck deck mazo que se va a repartir
results array array de resultados que llevará el recuento del número de veces que ha salido cada jugada.
numberHands int número de manos a repartir.
numberCardsPerHand int número de cartas que tendrá cada mano repartida
handPossibilities array todas las jugadas posibles de una mano.
